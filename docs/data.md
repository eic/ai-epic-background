## Conventions

- **FULL** — DD4hep simulation output (sim-level).
- **RECO** — EICrecon reconstruction output (reco-level).
- Entries are paired when a FULL and a RECO dataset share the same
  path tail; FULL is listed first.
- `main/` branch / CI / Test entries are filtered out — only versioned
  campaigns are shown.
- `epic_craterlake` is the only detector configuration and is omitted
  from the rendered paths' display group.
